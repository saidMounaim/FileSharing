# FileSharing

File sharing API Using Node js & Express js

## Features:

- File sharing

## Technology Stack:

- Node js
- Express Js
- MongoDB

## Default urls:

- Upload File : <br/>
  localhost:5000/api/upload
- Get All Files : <br/>
  localhost:5000/api/files
- Download File : <br/>
  localhost:5000/api/download/:fileId

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run server

```
